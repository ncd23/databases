APIs

HTTP

When people talk about APIs, they usually mean RESTful web services which use a combination of HTTP methods (verbs):

  • GET
  • POST
  • PUT
  • DELETE

An example of HTTP in action:

telnet en.wikipedia.org 80
GET /wiki/Main_Page http/1.1
Host: en.wikipedia.org

HTTP Status Codes:

  • 2xx Success
    • 200 OK
  • 3xx Redirection
    • 304 Not Modified
  • 4xx
    • 401 Not Authorized (HTTP Auth)
    • 403 Forbidden (App Auth)
    • 404 Not Found
  • 5xx Server Error
