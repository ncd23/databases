Indexes, Data types, and other RDBMS Capabilities

Indexes

Indexes can be a very complex topic. In summary, Oracle and PostgreSQL provide many more indexing options than the other databases discussed here.

Datatypes

All of the databases in this list support the same basic set of numeric (INTEGER and FLOAT), string (VARCHAR), and binary (BLOB) types, and in almost all cases the specific names will be the same. Oracle is the exception to this rule.

Database Capabilities

This includes SQL language features like Union, Inner/Outer Joins, Blobs, etc. The databases in this list are largely similar where capabilities are concerned.

Other capabilities in this category would include: triggers, functions, and procedures. These are supported by all of the databases listed here except for MS Access (none of the above) and SQLite (no functions and no procedures).
